Welcome to Node.js!

If you see this page, the Node.js® Express server is successfully installed and working.

The source code for this project is available at Guus / NodeJS Avans 1.4 · GitLab.

Usage :

  • [GET] /users : Returns JSON Array containing all existing users
  • [GET] /users/$id : Returns information about a given user
  • [GET] /users/random : Returns information about a random user
  • [PUT] /user : Placeholder for writing new users
  • [DELETE] /user : Placeholder for deleting users

Primary packages : Nodemon (Development), Express (Routing)
