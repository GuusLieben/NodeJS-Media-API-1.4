Welcome to Node.js!

If you see this page, the Node.js® Express server is successfully installed and working.

The source code for this project is available at Guus / NodeJS Avans 1.4 · GitLab. The base url is http://node.studyhost.nl:82/api

Usage :

Movies
  • [GET] /movies : Returns JSON Array containing all existing movies
  • [GET] /movies/:movieId : Returns information about a given movie
  • [POST] /movies : Creates a new movie and returns the movieId
  • [DELETE] /movies/:movieId : Deletes a movie with the given movieId
Users
  • [GET] /register : Returns JSON Array containing all existing users
  • [GET] /register/random : Returns information about a random user
  • [GET] /register/:userId : Returns information about a given user
  • [POST] /register : Creates a new user and returns the userId

Primary packages : Nodemon (Development), Express (Routing)
